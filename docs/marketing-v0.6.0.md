## 小红书标题

让 Codex 的状态，安静地待在 Mac 桌面上

## 小红书短文

每天打开 Codex，我最想第一眼看到的其实只有三件事：还剩多少额度、什么时候重置、任务满载后能不能自己继续。

所以这次我把 **Codex Helper** 彻底重新设计了一遍：

- 新的「状态轨道」是一条安静的横向信息带，剩余额度、重置倒计时和自动重试状态都在同一眼里；
- 主页面改成舒展的额度总览和扁平双栏布局，不再堆一排设置卡片；
- 背景始终保持中性，只让额度圆环轻轻变色：充足是蓝绿色，需要留意是琥珀色，接近用尽是粉红色；
- 遇到模型满载时，可以安全地回到原来的 Codex 任务继续重试；
- 支持中英文、自动更新、官方 Codex 动态和文档入口。

目前只支持 **macOS 13+**。项目公开在 GitHub，MIT License；不读取你的 Token，也不保存对话内容。

GitHub：<https://github.com/makerjackie/codex-helper>

## English short copy

**A quiet status rail for Codex on your Mac.**

Codex Helper keeps remaining quota, reset timing, and guarded Auto Retry visible in one horizontal rail. Its neutral interface uses color only as a small quota signal, while the redesigned dashboard keeps controls, official Codex updates, and documentation on one clean page.
